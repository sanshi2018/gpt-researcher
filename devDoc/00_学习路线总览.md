# GPT-Researcher 深度学习路线总览

> 📌 本文档是 GPT-Researcher 项目的系统化学习路线指南，聚焦于 **记忆体管理、RAG、多Agent、上下文管理、Function Call** 五大核心主题。

## 项目概览

GPT-Researcher 是一个开源的自主深度研究代理，核心思想源自 [Plan-and-Solve](https://arxiv.org/abs/2305.04091) 和 [RAG](https://arxiv.org/abs/2005.11401) 论文。它通过 **Planner-Executor** 架构，实现自动化的研究报告生成。

### 核心设计理念
```
用户查询 → Agent选择 → 子查询规划 → 并行检索 → 内容抓取 → 上下文压缩 → 报告生成
```

## 📖 建议学习方法

1. **先通读 Phase 1**：理解整体架构，建立全局视野
2. **按顺序学习 Phase 2-7**：每个 Phase 都有"关键代码解读"和"设计模式分析"
3. **Phase 8 串讲收尾**：把所有知识点串联起来，形成完整认知
4. **动手实验**：在每个 Phase 的末尾都有建议的动手练习
5. **回顾对比**：学完后回看项目源码，巩固理解
